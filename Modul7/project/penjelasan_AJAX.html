AJAX & jQuery

AJAX (Asynchronous JavaScript and XML) adalah teknik yang digunakan untuk mengambil data dari server tanpa harus me-reload halaman.

Contoh penggunaan:

  • Chat muncul tanpa refresh
  • Like / komentar langsung update

jQuery adalah library JavaScript yang mempermudah penulisan kode agar lebih singkat dan mudah dipahami.

Fungsi jQuery:

  • Memanipulasi HTML
  • Mengatur event (klik, hover)
  • Membuat animasi
  • Menggunakan AJAX

Cara Menggunakan jQuery:

1. Tambahkan library jQuery ke dalam HTML
2. Pilih elemen menggunakan selector (contoh: id atau class)
3. Tambahkan event seperti klik
4. Manipulasi elemen (ubah teks, sembunyikan, dll)
5. Gunakan AJAX untuk mengambil data tanpa reload
